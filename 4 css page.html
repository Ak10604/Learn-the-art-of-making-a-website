CSS Basics Selectors Properties Layouts

CSS Basics

CSS (Cascading Style Sheets) is used to style HTML documents. You can change colors, fonts, layouts, and much more!

To include CSS in an HTML document, you can use:

<link rel="stylesheet" href="styles.css">

CSS Selectors

Select the elements you want to style using various selectors:

  • element - Selects all elements of that type (e.g., p selects all paragraphs).
  • .class - Selects all elements with that class (e.g., .myClass).
  • #id - Selects a single element with that ID (e.g., #myId).

CSS Properties

CSS properties allow you to change the appearance of elements:

  • color - Changes the text color.
  • background-color - Changes the background color.
  • font-size - Changes the size of the text.
  • margin - Adjusts the space around elements.
  • padding - Adjusts the space inside elements.

CSS Layouts

Use CSS to create different layouts:

  • Flexbox - A layout model for responsive designs.
  • Grid - A two-dimensional layout system.
  • Positioning - Control the placement of elements.
